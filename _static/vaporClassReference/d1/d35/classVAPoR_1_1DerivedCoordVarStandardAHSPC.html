Detailed Description

Convert a CF parameterless vertical coordinate to an Ocean s-coordinate, generic form 1 or 2.

This derived class converts a dimensionless sigma coordinate variable to either a Ocean s-coordinate, generic form 1 or 2

See also
http://cfconventions.org/
